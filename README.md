
  # VirtualHR

  ## Description
  Virtual HR is a terminal application created as a database management application which handles employee information.

  ## Installation
  - Clone the git repo on a local folder
- Navigate to the local folder with your preferred terminal application
- Run `npm i` (REQUIRES NODE JS)


  ## Usage
  - Run `npm run seed`
- Fill your MySQL login info to seed the database
- Run `npm start`
- Fill your MySQL login info
- Navigate the database and add entries
- [View usage video]
